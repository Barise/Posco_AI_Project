# 포스코 AI 프로젝트 'POPANG'
### 딥러닝을 활용한 창고 관리 시스템(WMS) 개선으로 유통업계의 창고 비용 절감

#### 기한: 7월 27일부터 ~ 8월 20일까지(4주)

- Github : https://github.com/Barise/Posco_AI_Project

#### 기술스택:

- 프론트 : React(Native)
- 백엔드 : Flask(Python)

#### 팀 구분:

- 팀장 : 강태훈
- 팀원 : 이지현, 최형찬, 김태호

#### 서비스 개요

- 딥러닝을 활용한 창고 관리 시스템
- CNN을 통해 선반(bin) 속 재고 개수와 남는 공간을 파악한다.
