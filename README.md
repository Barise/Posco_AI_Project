# POSCO 청년 AI·Big Data 아카데미 AI 프로젝트
> 딥러닝을 활용한 창고 관리 시스템(WMS) 개선으로 유통업계의 창고 비용 절감  

CNN(Inception V3)를 통해 선반(bin) 속 재고 개수와 남는 공간을 파악하는 창고 관리 시스템입니다.
